# Asset Upload Instructions

Please upload the following assets to the `public/` directory for the application to function correctly:

1.  **totodile-card.png**
    -   This image will be displayed inside the green screen area of the app.
    -   Upload to: `public/totodile-card.png`

2.  **bg-music.mp3**
    -   This audio file will play as background music (looped).
    -   Upload to: `public/bg-music.mp3`

Without these files, the image placeholder will be broken, and the background music will not play.
