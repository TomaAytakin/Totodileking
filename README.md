# Totodile Treasury Tracker

A React-based web application that tracks the treasury value and token statistics for the Totodile King community on Solana.

## Tech Stack

- **Frontend:** React, Tailwind CSS
- **Backend:** Google Cloud Functions
- **Blockchain:** Solana Web3.js
- **Icons:** Lucide React

## Features

- **Live Token Price:** Fetches real-time price from Dexscreener.
- **Treasury Balance:** Tracks the community treasury balance on Solana.
- **Card Price Integration:** Fetches card price data from a custom API.
- **Gameboy UI:** Styled with a retro Gameboy/Pokedex aesthetic.
